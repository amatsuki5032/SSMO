# M3-5b: NPC兵士AI行動

読むファイル: NPCSoldier.cs, NPCSpawner.cs, GameConfig.cs
変更ファイル: NPCSoldier.cs, GameConfig.cs

NPC兵士の基本AI行動を実装する。

1. NPCSoldier.cs にAI行動を追加：
   - 基本ループ: 移動 → 敵検出 → 攻撃 → 移動
   - 移動: 最寄りの敵拠点方向へ NPC_MOVE_SPEED で直進
   - 敵検出: 半径 NPC_DETECT_RANGE 内の敵（プレイヤー or 敵NPC）を検出
   - 攻撃: 検出した敵に向かって移動 → 攻撃範囲内で自動攻撃
   - 攻撃はN1相当（単発、NPC_ATK_INTERVAL 間隔）
   - 攻撃判定はサーバー権威（OverlapSphere）
   - 敵NPC同士も戦闘する

2. GameConfig.cs に追加：
   - NPC_DETECT_RANGE = 8f
   - NPC_ATTACK_RANGE = 1.5f
   - NPC_ATK_INTERVAL = 1.5f（攻撃間隔）

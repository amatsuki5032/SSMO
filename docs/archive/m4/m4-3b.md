# M4-3b: エボリューション攻撃

読むファイル: ComboSystem.cs, MusouGauge.cs, HitboxData.cs, WeaponData.cs
変更ファイル: ComboSystem.cs, HitboxData.cs, CharacterState.cs

エボリューション攻撃（E6-E9）を実装する。

1. 発動条件:
   - 連撃強化レベル3
   - 無双ゲージMAX
   - N5 の後に自動的にE6に移行（□連打で継続）

2. CharacterState.cs：
   - 必要なら Evolution ステート追加（or 既存 Attack ステートで comboStep 7-10 として管理）

3. ComboSystem.cs：
   - N5 完了時に連撃Lv3 + ゲージMAX判定 → E6 に遷移
   - E6→E7→E8→E9 のコンボチェーン
   - **ゲージは消費しない**
   - E は**チャージ攻撃扱い**（アーマー3段階を崩せる = AttackLevel.Charge）

4. HitboxData.cs / WeaponData.cs：
   - E6-E9 のヒットボックスデータ追加
   - combat-spec の倍率: E6=1.3, E7=1.5, E8=1.7, E9=2.0（仮値）
   - E6=N1モーション流用, E7=N2, E8=N3, E9=C4（フィニッシュ）

5. コンボ全体フロー確認:
   N1→N2→N3→N4→N5→E6→E7→E8→E9

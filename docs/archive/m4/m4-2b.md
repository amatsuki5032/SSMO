# M4-2b: 炎・風の状態異常

読むファイル: CharacterStateMachine.cs, CharacterState.cs, GameConfig.cs, ElementSystem.cs
変更ファイル: CharacterStateMachine.cs, GameConfig.cs, 新規 Scripts/Combat/StatusEffectManager.cs

炎（燃焼）と風（鈍足）の状態異常を実装する。

1. 新規 Scripts/Combat/StatusEffectManager.cs：
   - NetworkBehaviour, サーバー権威
   - 状態異常の付与・解除・tick処理を一元管理
   - 各状態異常の残り時間をサーバーで管理

2. 炎（燃焼 / Burn）：
   - チャージ攻撃ヒット時に付与
   - 持続ダメージ（毎秒）
   - **HPを0にはできない**（最低HP1で止める）
   - 受け身を取るまで空中で燃焼継続
   - 地上で行動可能になったら解除
   - CharacterStateMachine の StatusEffect.Burn フラグと連携

3. 風（鈍足 / Slow）：
   - チャージ攻撃ヒット時に付与
   - 移動速度低下（50%仮値）
   - **ジャンプ不可**
   - 時間経過で解除（仮: 5秒）
   - CharacterStateMachine の StatusEffect.Slow フラグと連携
   - PlayerMovement で Slow フラグ時の移動速度制限

4. GameConfig.cs に追加：
   - BURN_DAMAGE_PER_SEC = 10f（仮値）
   - BURN_TICK_INTERVAL = 0.5f
   - SLOW_DURATION = 5f（仮値）
   - SLOW_SPEED_MULT = 0.5f（仮値）

# M4-4a: 仙箪ドロップ＆拾い

読むファイル: NPCSoldier.cs, GameConfig.cs, CharacterState.cs
変更ファイル: GameConfig.cs, NPCSoldier.cs, 新規 Scripts/Combat/SentanItem.cs

仙箪アイテムのドロップと拾い判定を実装する。

1. 新規 Scripts/Combat/SentanItem.cs：
   - NetworkBehaviour, サーバー権威
   - ドロップされたアイテムの位置を同期（NetworkVariable<Vector3>）
   - 拾い判定: プレイヤーが一定距離内に入ると自動取得（OnTriggerEnter）
   - 取得時にプレイヤーの仙箪カウントを+1
   - 一定時間で消滅（仮: 30秒）
   - Prefab化が必要（NetworkObject付き）

2. NPCSoldier.cs：
   - NPC死亡時に仙箪アイテムをスポーン（サーバー側）
   - ドロップ確率（仮: 100%。バランスは後で調整）

3. GameConfig.cs に追加：
   - SENTAN_DROP_RATE = 1.0f
   - SENTAN_PICKUP_RADIUS = 2.0f
   - SENTAN_LIFETIME = 30f
   - SENTAN_REQUIRED_FOR_ENHANCE = 7（強化に必要な仙箪数）

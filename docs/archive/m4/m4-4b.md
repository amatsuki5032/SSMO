# M4-4b: 強化リングUI＆発動

読むファイル: SentanItem.cs, ComboSystem.cs, GameConfig.cs, WeaponData.cs
変更ファイル: GameConfig.cs, 新規 Scripts/Combat/EnhancementRing.cs, 新規 Scripts/UI/EnhancementRingHUD.cs

仙箪強化リングシステムを実装する。

1. 新規 Scripts/Combat/EnhancementRing.cs：
   - NetworkBehaviour, サーバー権威
   - 7スロットのリング管理:
     - スロット1-3: 武器依存（攻撃UP/防御UP/移動UP等）
     - スロット4: 副将強化（固定、M4ではダミー効果）
     - スロット5-6: 武器依存
     - スロット7: 連撃強化（固定）
   - 仙箪を7個集めるとリングが回転開始
   - NetworkVariable<int> で現在の仙箪カウントを同期
   - NetworkVariable<int> で現在のリング位置を同期
   - R1（Eキー）でスロット発動
   - 発動時にスロット効果を適用:
     - 連撃強化 → ComboSystem.EnhanceCombo()
     - 攻撃UP → ATKバフ（仮: +10%）
     - 防御UP → DEFバフ（仮: +10%）
   - 死亡時に全強化リセット

2. 新規 Scripts/UI/EnhancementRingHUD.cs：
   - MonoBehaviour（クライアント専用UI）
   - 画面に仙箪カウント表示
   - リング回転中はスロット表示

3. GameConfig.cs に追加：
   - SENTAN_SLOTS = 7
   - RING_ROTATION_SPEED = 2f（回転速度）
   - ATK_BUFF_PER_ENHANCE = 0.1f
   - DEF_BUFF_PER_ENHANCE = 0.1f

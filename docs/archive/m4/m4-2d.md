# M4-2d: 斬属性

読むファイル: DamageCalculator.cs, HitboxSystem.cs, MusouGauge.cs, ElementSystem.cs
変更ファイル: DamageCalculator.cs, HitboxSystem.cs, MusouGauge.cs

斬属性の特殊ルールを実装する。

1. DamageCalculator.cs：
   - 斬属性の場合 **DEF = 0 として計算**（防御力無視）
   - 最終ダメージ = max(通常計算ダメージ, 斬固定値)
   - 斬固定値はレベル依存（Lv1:10, Lv2:20, Lv3:30, Lv4:40）
   - Calculate() の戻り値 DamageResult に斬属性フラグを反映

2. HitboxSystem.cs：
   - 斬属性ダメージ時、被弾者の **HP と 無双ゲージ両方** にダメージ
   - **攻撃側の無双ゲージも減少**（デメリット）
   - DamageResult の MusouDamage / AttackerMusouCost を使用

3. MusouGauge.cs：
   - ConsumeGauge() を斬属性の攻撃側ゲージ減少にも使用
   - 被弾側のゲージ減少用の ReduceGauge(float) を追加（TakeDamage的な用途）

4. 斬属性のバランス確認:
   - 高リスク高リターン（攻撃側も無双減少）
   - 状態異常なし（他属性との差別化）

# M4-2c: 氷・雷の状態異常

読むファイル: CharacterStateMachine.cs, CharacterState.cs, StatusEffectManager.cs, ReactionSystem.cs
変更ファイル: CharacterStateMachine.cs, StatusEffectManager.cs, ReactionSystem.cs, GameConfig.cs

氷（凍結）と雷（感電+気絶）の状態異常を実装する。

1. 氷（凍結 / Freeze）：
   - **確率発動**（仮: 30%）
   - ヒット時に判定 → 凍結成功なら Freeze ステートへ強制遷移
   - **約2秒間完全行動不能**
   - 凍結解除モーション後にIdle
   - **無双で脱出不可**（Freezeステートでは無双入力を受け付けない）
   - CharacterStateMachine に Freeze ステートの遷移ルール追加

2. 雷（感電 / Electrified + 気絶 / Stun）：
   - 感電:
     - **受け身が取れなくなる**（AirRecover遷移を無効化）
     - 空中・地上どちらでも付与
     - 解除条件: 攻撃を受けなければ2秒で解除 / 10コンボで解除 / ダウンで解除
     - StatusEffect.Electrified フラグで管理
   - 気絶:
     - **地上で雷属性を受けた時のみ発動**（空中では気絶しない）
     - Stun ステートへ遷移（約3秒行動不可）
     - **無双で脱出不可**
     - 気絶中に攻撃を受けると浮く（Launch遷移）
     - 攻撃モーションごとに気絶付与の有無を設定（仮: 全雷チャージに付与）

3. ReactionSystem.cs：
   - 感電中は AirRecover 遷移をブロック
   - 感電のコンボカウント追跡

4. GameConfig.cs に追加：
   - FREEZE_PROBABILITY = 0.3f（仮値）
   - FREEZE_DURATION = 2.0f
   - ELECTRIFIED_DURATION = 2.0f（攻撃なし時の自然解除）
   - ELECTRIFIED_COMBO_LIMIT = 10
   - STUN_DURATION = 3.0f

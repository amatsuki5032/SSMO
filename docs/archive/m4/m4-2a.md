# M4-2a: 属性システム基盤

読むファイル: CharacterState.cs, DamageCalculator.cs, HitboxSystem.cs, GameConfig.cs
変更ファイル: GameConfig.cs, DamageCalculator.cs, 新規 Scripts/Combat/ElementSystem.cs

属性システムの基盤を作成する。属性はチャージ攻撃にのみ乗る。

1. 新規 Scripts/Combat/ElementSystem.cs：
   - NetworkBehaviour, サーバー権威
   - プレイヤーの装備属性を管理: NetworkVariable<ElementType>, NetworkVariable<int> (属性レベル1-4)
   - チャージ攻撃時に属性を付与するか判定
   - 属性レベルに応じた倍率計算は DamageCalculator に委譲

2. DamageCalculator.cs：
   - 既存の GetElementDamageMultiplier() を確認・拡張
   - combat-spec セクション14 の属性レベル倍率表:
     - 炎: 0.175/Lv（Lv1: ×1.175, Lv4: ×1.70）
     - 氷: 0.25/Lv（Lv1: ×1.25, Lv4: ×2.00）
     - 雷: 0.50/Lv（Lv1: ×1.50, Lv4: ×3.00）
     - 風: 0.50/Lv（Lv1: ×1.50, Lv4: ×3.00）
     - 斬: 最低保証ダメージ（Lv1:10, Lv2:20, Lv3:30, Lv4:40）

3. GameConfig.cs に追加：
   - ELEMENT_FIRE_MULT_PER_LV = 0.175f
   - ELEMENT_ICE_MULT_PER_LV = 0.25f
   - ELEMENT_THUNDER_MULT_PER_LV = 0.50f
   - ELEMENT_WIND_MULT_PER_LV = 0.50f
   - SLASH_MIN_DAMAGE = new int[] { 0, 10, 20, 30, 40 }

4. HitboxSystem.cs：
   - ダメージ計算時に ElementSystem から属性情報を取得
   - チャージ攻撃の場合のみ属性倍率を適用

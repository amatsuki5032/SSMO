## 2. 移動 & ターゲティング

### 移動速度
- 武器種ごとに基本移動速度が異なる
- 装備アイテム・武器改造・拠点バフ等で変動

### 攻撃中の方向転換
- **コンボ中に方向転換可能**（スティック入力で向き調整）
- 向き調整の精度がプレイヤースキル

### 微弱ロックオン
- 明示的なロックオン操作はない
- **敵が1人の場合、入力なしでもその敵に向かって攻撃する**（自動追尾）
- **周囲に雑兵がいると雑兵に向き直ってしまう** → 手動での方向調整が必要

### ヒットストップ
- **なし**。攻撃ヒット時にゲーム時間は止まらない
- 常時戦闘が流れるスピード感を重視

## 3. 通常攻撃コンボ (N)

### コンボ段数は成長する
- **無強化**: N1〜N4 まで（4段）
- **連撃強化1回**: N5 まで（5段）
- **連撃強化2回**: N6 まで（6段）
- **連撃強化3回 + 無双ゲージMAX**: エボリューション攻撃で E6〜E9 まで（ゲージ消費なし）

### コンボ制御
- **コンボ受付ウィンドウ**: 各攻撃モーションの最後30%のフレーム
- **先行入力バッファ**: 150ms（入力を最大150ms先に保持）
- サーバーがコンボ段数と連撃強化レベルを追跡し不正な連鎖を防止

### モーション倍率（例: 大剣）※仮値

| 技 | 倍率 | 説明 |
|----|------|------|
| N1 | 0.8 | 初段 |
| N2 | 0.9 | |
| N3 | 1.0 | |
| N4 | 1.1 | 無強化の最終段 |
| N5 | 1.2 | 連撃強化1で解放 |
| N6 | 1.5 | 連撃強化2で解放 |

### エボリューション攻撃 (E)
- 連撃強化3回 + 無双ゲージMAX で N5 の後に E6〜E9 に移行
- 無双ゲージは消費しない
- **E はチャージ攻撃扱い**（アーマー3段階を崩せる）

| 技 | 倍率（仮） | モーション |
|----|-----------|----------|
| E6 | 1.3 | N1 のモーション流用 |
| E7 | 1.5 | N2 のモーション流用 |
| E8 | 1.7 | N3 のモーション流用 |
| E9 | 2.0 | C4 のモーション流用（フィニッシュ） |

コンボ全体: N1→N2→N3→N4→N5→E6→E7→E8→E9

## 4. チャージ攻撃 (C)

N[x] の途中で △ を押すと C[x] に派生。各 C 技は武器種ごとに固有。

| 技 | 入力 | 効果（公式準拠） | 倍率（仮） |
|----|------|----------------|-----------|
| C1 | △ | 武器ごとの強力な攻撃（単体） | 2.0 |
| C2 | N→△ | 打ち上げ（空中コンボ起点） | 1.5 |
| C3 | NN→△ | ラッシュ攻撃（△連打で追加ヒット） | 0.4×n hit |
| C4 | NNN→△ | 吹き飛ばし（巻き込み：飛んだ敵が他の敵にぶつかる） | 1.8 |
| C5 | NNNN→△ | チャージシュート（まとめて打ち上げ、C2の範囲版） | 1.3 |
| C6 | NNNNN→△ | 武器種固有の強力な連続攻撃 | 3.0 |

### C3 ラッシュの仕様
- C3 発動後、△を連打するとラッシュ（追加攻撃）が続く
- ラッシュ終了後にアイドルに戻る

## 5. ダッシュ攻撃 (D)

### 発動条件
- **一定時間（仮: 1.5秒）以上連続移動した状態で □ を押す**
- 移動距離ではなく移動時間で判定（鈍足でもD発動可能）

### ダッシュ攻撃の派生

| 入力 | 技 | 説明 |
|------|-----|------|
| 一定時間移動→□ | D | ダッシュ攻撃（N1とは別モーション） |
| D→□連打 | Dラッシュ | 追加攻撃（ラッシュ） |
| D→L2 | DBC | ダッシュブレイクチャージ |
| D→△ | なし（将来拡張枠） | DC として追加可能 |

### 注意
- D は N コンボとは別系統で完結する
- D から N コンボには移行しない
- D→ラッシュは □ 連打で継続

## 6. ジャンプ

- × で垂直ジャンプ。左スティック同時押しでその方向に飛ぶ
- **ジャンプ中は方向転換不可**
- **2段ジャンプなし**
- **ジャンプ中ガード不可**
- ジャンプの高さ・滞空時間は武器種ごとに異なる（装備アイテムでも増減）
- **無敵フレームあり**（離陸時の数フレーム）

### ジャンプ中の攻撃

| 入力 | 技名 | 説明 |
|------|------|------|
| ジャンプ中 □ | ジャンプ攻撃 (JA) | 空中通常攻撃 |
| ジャンプ中 △ | ジャンプチャージ (JC) | 空中チャージ攻撃 |

### ジャンプパラメータ（仮値、武器種で変動）

| パラメータ | 仮値 | 備考 |
|-----------|------|------|
| ジャンプ高さ | 3m | 武器種・装備で変動 |
| 滞空時間 | 0.6s | 武器種・装備で変動 |
| 離陸無敵 | 4F (67ms) | 仮値。テストで調整 |

## 7. 打ち上げ & 空中コンボ

### 打ち上げの流れ
1. C2 / C5 等の打ち上げ攻撃がヒット
2. 被弾者が空中に浮く
3. **受け身不能時間**（約1秒。攻撃モーションごとに設定）
4. 攻撃側はジャンプして JA / JC / N で追撃
5. 受け身不能時間が過ぎると被弾者は受け身で脱出可能

### 空中コンボのルール
- **空中でのヒット数上限なし**（無限につなげられる）
- ただし受け身が取れれば脱出可能
- **空中補正: ダメージ÷2**（空中で受けたダメージは半減）
- C2 以外にも打ち上げ効果を持つ技あり（C5、武器種固有技）
- 空中→地面は叩きつけ or 吹き飛ばし

### 受け身（エリアルリカバリー）
- 受け身不能時間が終了後、× で受け身
- 受け身成功時は短い無敵時間あり

## 8. ガード

### 通常ガード (G)
- L1 押しっぱなしでガード状態
- **正面180度の攻撃を防御**
- ダメージ80%カット（ガード時は20%ダメージ）
- **ガードブレイクなし**（正面ガードは絶対崩れない）
- **ガード移動可能**（L1+方向キーで正面を向いたまま移動。正面・真横・後方のみ）
- **無双乱舞はガード可能**

### ガードを崩す手段

| 方法 | 説明 |
|------|------|
| めくり | 正面以外（側面・背面）から攻撃を当てる |
| 反転ガード | 防御側がガード解除→向き直し→再ガードで対応 |

### ガード不可技
- 基本的にガード不可技はなし
- 一部の特殊技がガード不可の可能性あり（要検討）

---

## 9. エレメンタルガード (EG)

### 発動条件
- **無双ゲージが一定以上溜まっていること**

### 発動手順
1. L1（ガード状態に入る）
2. ガード中に △（チャージ）を **約1秒押し込み** → EG完成
3. L1 + △ を押しっぱなしで維持
4. **EG準備中もガードは有効**

### EG発動（カウンター）
- EG完成中に敵が攻撃してくると **EG発動**
- 敵は属性に応じた反撃を受ける
- **発動時に無双ゲージを一定量消費**

### EG維持コスト
- EG状態中は **無双ゲージが少しずつ減少**

### EG解除条件
- ガードを解除する
- △（チャージ）ボタンを離す
- 無双ゲージが0になる

### EG の弱点
- **EG はめくりに対応できない**（反転ガードでないとめくられる）

### 属性別 EG効果

| 属性 | EG 発動時効果 |
|------|-------------|
| 無属性 | 吹き飛ばし（ダメージなし） |
| 炎 | 燃焼 + その場で崩れ落ちダウン |
| 氷 | その場で凍結 |
| 雷 | 感電 + 気絶 |
| 風 | 鈍足付与 + 強吹き飛ばし |
| 斬 | 吹き飛ばし + 割合ダメージ + 無双ゲージ減少 |

## 10. 無双ゲージシステム

### 無双ゲージの初期値
- **戦闘開始時: 0（空）**
- **リスポーン時: MAX（満タン）**

### 無双ゲージの溜め方

| 方法 | 詳細 |
|------|------|
| 攻撃を当てる | 攻撃倍率が大きいほど多く溜まる |
| 攻撃を受ける | 攻撃倍率が大きいほど多く溜まる |
| 無双チャージ | ○長押しでその場停止してゲージを溜める |
| 究極強化効果 | 特定の究極効果中にガード成功で溜まる |
| 装備アイテム | 自動で溜まるアイテムあり |
| 現地アイテム | 戦場にポップするアイテムで一瞬回復 |
| 無双の鍛錬 | 鍛錬量が多いほど溜まりやすい |

### 無双チャージ
- ○ 長押し（ゲージがMAX未満の時）
- **その場に停止**して溜める（移動不可）
- **ガードなし**（通常状態と同じ被弾扱い）
- モーションはガードモーションと同じ（専用モーション検討可）

### 無双ゲージの消費

| 行動 | 消費 |
|------|------|
| 無双乱舞 | 全消費（MAX→0） |
| EG維持 | 少しずつ減少 |
| EG発動（カウンター成功） | 一定量消費 |
| 斬属性で攻撃を受ける | 一定量減少 |

## 11. アーマーシステム

### アーマー段階（5段階）

| 段階 | 名称 | 耐性 | 付与方法 |
|------|------|------|---------|
| 1 | 通常 | なし（全てのけぞる） | デフォルト |
| 2 | 矢耐性 | 雑魚の矢でのけぞらない | 装備/究極強化 |
| 3 | N耐性 | 通常攻撃でのけぞらない | 装備/究極強化/特定モーション中 |
| 4 (SA) | スーパーアーマー | チャージでものけぞらない | 装備/究極強化 |
| 5 (HA) | ハイパーアーマー | 無双でものけぞらない | 装備/究極強化 |

### 攻撃レベル（のけぞらせる力）

| 攻撃レベル | 該当する攻撃 |
|-----------|------------|
| 1 | 雑魚の矢 |
| 2 | 通常攻撃 (N) |
| 3 | チャージ攻撃 (C) / エボリューション (E) |
| 4 | 無双乱舞 |
| 5 | HA貫通（要検討） |

### ルール
- **ダメージは常に通る**（アーマーはのけぞり無効化のみ）
- 攻撃レベル > アーマー段階 → のけぞる
- 攻撃レベル ≤ アーマー段階 → のけぞらない（ダメージは受ける）
- **無双乱舞中は無敵**（アーマーとは別。ダメージ自体を受けない）

## 12. 被弾リアクション

### のけぞり（ヒットスタン）
- 攻撃を受けた時の基本リアクション
- **後ろに体が傾くもの**と**前に傾くもの**の2種がある（コンボ猶予への影響は未確認）
- 一定時間行動不能、この間に追撃→またのけぞり→コンボ成立
- **のけぞり中に無双乱舞で脱出可能**

### ダウン系（4種類）

| 種類 | 無双で脱出 | 追撃を受けた時 | モーション長 | トリガー例 |
|------|-----------|-------------|------------|-----------|
| **前のめりダウン** | 不可 | **のけぞりに遷移** | 普通 | 吹き飛ばし(C4)等 |
| **崩れ落ちダウン** | 不可 | **浮く** | **長い** | 炎燃焼EG等 |
| **仰向けダウン** | 不可 | **浮く** | **短い** | 叩きつけ/空中落下 |
| **気絶** | **不可（無双も不可）** | **浮く** | 約3秒 | 地上で雷属性/特定技 |

### 重要: ダウン→前のめりダウン遷移（地上ハメルート）
```
崩れ落ちダウン →(攻撃)→ 浮く → 空中コンボ（通常ルート）
崩れ落ちダウン →(前のめりD技)→ 前のめりダウン →(攻撃)→ のけぞり → 地上コンボ継続
仰向けダウン  →(前のめりD技)→ 前のめりダウン →(攻撃)→ のけぞり → 地上コンボ継続
気絶         →(攻撃)→ 浮く → 空中コンボ
```

- **他のダウン状態の敵に対しても前のめりダウンにできる**
- これにより空中コンボではなく地上ハメが可能になるルートが存在

### ダウン共通ルール
- **ダウン中は追い打ち不可**（ダウン中の敵には攻撃が当たらない ※前のめりダウンを除く）
- **起き上がりモーションは必ず発生**（スキップ不可、全種共通）
- **起き上がり中は無敵**（仮。未確定事項）

## 13. 状態異常

### 燃焼（炎属性）
- 持続ダメージ。**HPを0にはできない**
- 受け身を取るまで燃焼継続
- まれに地上でも燃焼状態になる。行動可能になったら解除
- 天候が雨・雪の場合はダメージ減少

### 凍結（氷属性）
- **確率発動**
- **約2秒間完全行動不能**
- 凍結解除モーション後に行動可能
- 天候が雪の場合は凍結時間延長

### 感電（雷属性）
- **受け身が取れなくなる**
- 空中・地上どちらでも付与される
- 解除条件: **攻撃を受けなければ2秒で解除** / **10コンボで解除** / **ダウンしたら解除**
- 一定コンボで感電解除後、そのコンボ中は再発動しない

### 気絶（雷属性 + 地上ヒット時のみ）
- **地上で雷属性を受けた時に感電と同時に発動**
- **空中では気絶しない**（ただし気絶攻撃を空中で受けると通常と異なる浮き量）
- その場で約3秒行動不可
- 攻撃モーションごとに気絶付与の有無が設定されている
- 気絶中に攻撃を受けると浮く

### 鈍足（風属性）
- **移動速度低下 + ジャンプ不可**
- 時間経過で解除
- 天候が雨・雪の場合は属性ダメージ追加

## 14. 属性システム

### 属性一覧（5種 + 無属性）
- **属性同士の相性なし**（有利不利はない）
- **属性はチャージ攻撃にのみ乗る**（通常攻撃には乗らない）
- 究極強化「鳳凰」中は全攻撃に属性効果

### 属性の付与方法
- 武器に属性錬成（+1レベル）
- 装備アイテムの属性玉（+1〜2レベル）
- 味方拠点内にいると拠点属性が付く（鼎が残っている場合のみ）
- 消費アイテムの属性が優先

### 属性レベルと倍率

属性が乗るチャージ攻撃時に、属性レベルに応じて攻撃倍率に乗算。

| 属性 | 倍数/Lv | Lv1 | Lv2 | Lv3 | Lv4 |
|------|---------|-----|-----|-----|-----|
| 炎 | 0.175 | ×1.175 | ×1.35 | ×1.525 | ×1.70 |
| 氷 | 0.25 | ×1.25 | ×1.50 | ×1.75 | ×2.00 |
| 雷 | 0.50 | ×1.50 | ×2.00 | ×2.50 | ×3.00 |
| 風 | 0.50 | ×1.50 | ×2.00 | ×2.50 | ×3.00 |
| 斬 | - | 最低保証10 | 最低保証20 | 最低保証30 | 最低保証40 |

### 属性倍率の計算
```
属性付き倍率 = モーション倍率 × (1.0 + 属性倍数 × レベル)
例: 大剣C1(2.0) × 炎Lv3(×1.525) = 3.05
```

### 斬属性の特殊ルール
- **防御力無視**（DEF=0 として計算）
- **HP + 無双ゲージ両方にダメージ**
- **攻撃側の無双ゲージも減少**（デメリット）
- ダメージは **max(通常計算ダメージ, 斬固定値)** で最低保証
- 兵士は即死の可能性あり

### 属性バランスの棲み分け

| 属性 | 倍率 | 状態異常 | 特徴 |
|------|------|---------|------|
| 炎 | 低 | 燃焼（持続ダメ） | 継続ダメージ型 |
| 氷 | 中 | 凍結（行動不能） | コンボ起点型 |
| 雷 | 高 | 感電+気絶 | 高火力+拘束 |
| 風 | 高 | 鈍足+ジャンプ不可 | 高火力+機動力奪取 |
| 斬 | 固定 | なし | 対硬敵+無双管理（諸刃） |

## 15. 無双乱舞

### 通常無双乱舞
- ○ でゲージMAX時に発動
- 連続攻撃 + **無敵**
- **ガード可能**（めくりでのみ通る）
- 倍率: 0.3×15hit + Finisher 4.0（仮）

### 真・無双乱舞
- **HPが赤（20%以下）の時に無双乱舞を発動**
- 攻撃倍率が変化
- **炎属性が付与される**

### 激・無双乱舞
- 味方（指揮官/副将/無双武将/NPC指揮官）が近くで無双乱舞発動中に自分も発動
- 電撃で結ばれると発動可能のサイン
- 強さは真・無双乱舞と同等

## 16. HP & ダメージ計算式

### 根性補正（体力ゲージシステム）

| HP帯 | ゲージ色 | ダメージ補正 |
|------|---------|------------|
| 50〜100% | 青 | 等倍（÷1） |
| 20〜50% | 黄 | **÷1.5**（33%カット） |
| 20%以下 | 赤 | **÷2**（50%カット） |

### ダメージ計算式（★ サーバー側のみで実行 ★）

```
1. 攻撃倍率 = モーション倍率 × 属性倍率（属性が乗る攻撃の場合）
2. 基礎ダメージ = ATK × 攻撃倍率
3. 防御計算 = 基礎ダメージ × (100 / (100 + DEF))
   ※斬属性の場合は DEF = 0 として計算
4. 空中補正 = 空中被弾時は ÷2
5. 根性補正 = HP青:÷1 / HP黄:÷1.5 / HP赤:÷2
6. ガード補正 = ガード時 ×0.2 / 非ガード ×1.0
7. 最終ダメージ = 防御計算 ÷ 空中補正 ÷ 根性補正 × ガード補正
8. 斬保証 = 斬属性の場合 max(最終ダメージ, 斬固定値)
9. 最低ダメージ保証 = max(最終ダメージ, 1)
10. クリティカル = 5%確率で ×1.5（仮値）

★ 斬属性の場合: HP と無双ゲージ両方にダメージ
★ 斬属性の場合: 攻撃側の無双ゲージも減少
```

### 鍛錬システム（M4向け）

#### 概要
- 武器ごとに **鍛錬上限** がある（例: 24）
- 鍛錬ポイントを攻撃力/破壊力/防御力/体力/無双に振り分ける
- 各ステータスの1段階ごとに仙箪強化の上昇値が+1

#### 鍛錬コスト（三角数方式）
```
n段階のコスト = 1 + 2 + 3 + ... + n = n × (n+1) ÷ 2

例: 鍛錬上限24の武器
  攻撃5段階: 5×6÷2 = 15
  防御3段階: 3×4÷2 = 6
  体力2段階: 2×3÷2 = 3
  合計: 15 + 6 + 3 = 24（上限ぴったり）
```

#### ビルド例

| ビルド | 攻撃 | 破壊 | 防御 | 体力 | 無双 | 合計コスト |
|--------|------|------|------|------|------|-----------|
| 攻撃特化 | 5(15) | 0 | 0 | 3(6) | 1(1) | 22 |
| バランス | 3(6) | 0 | 3(6) | 3(6) | 3(6) | 24 |
| タンク | 1(1) | 0 | 5(15) | 3(6) | 1(1) | 23 |

- 高レベルほどコスパが悪い → 一点特化 vs バランスの選択が生まれる

#### 仙箪スロットとの連動
- 武器の仙箪強化スロット（7個中1〜3, 5〜6）に各ステータスが割り当てられる
- スロット順は武器種によって異なる
- 鍛錬段階が高いほど仙箪1個あたりの上昇値が大きい
- 未鍛錬のスロットは「????」（ランダム効果）

## 18. 仙箪強化システム

### 概要
- 戦闘中に敵兵を倒すと仙箪（せんたん）をドロップ
- 仙箪を拾うと強化リングのスロットが回転
- 目的のスロットが来たら R1 で発動

### 強化リング（7スロット）

| スロット | 種類 | 効果 |
|---------|------|------|
| 1〜3 | 武器依存 | 攻撃強化 / 防御強化 / 移動強化 等 |
| 4 | 副将強化（固定） | 副将能力UP + 副将スキル発動 |
| 5〜6 | 武器依存 | 攻撃強化 / 防御強化 / 移動強化 等 |
| 7 | 連撃強化（固定） | コンボ段数増加 |

### 連撃強化の重要性
- 1回: N5 解放
- 2回: N6 解放
- 3回: エボリューション攻撃解放（+無双ゲージMAX必要）
- **3回発動後の4回目以降 → 究極強化に変化**

### 強化リセット
- HPが0になって撤退すると **全強化リセット**
- 戦闘条件「撤退時強化：維持」なら保持

## 19. 究極強化

- 連撃強化3回発動後、さらに仙箪7個集めて発動
- **30秒間の超強力バフ**
- 武器種ごとに効果が異なる
- 覚醒玉アイテム装備時は無双覚醒（武将に変身）

## 23. 死亡 & リスポーン

### 死亡時の流れ
1. HPが0になる → Dead ステート
2. **即座にリスポーン**（待ち時間なし）
3. リスポーン地点で復活

### リスポーン地点の決定（交互制限）
- **1回目**: 最も近い自軍拠点にリスポーン
- **2回目**: 前回使った拠点は除外 → 次に近い自軍拠点
- **3回目**: 再び最寄り拠点が使用可能（前回の除外がリセット）
- **交互繰り返し**（同じ拠点に連続リスポーン不可）

### リスポーン対象拠点
- 絶対に敵に取られない固定拠点
- 奪い合える拠点のうち自軍が制圧中のもの

### 強化リセット
- リスポーン時に **仙箪強化が全リセット**
- 戦闘条件「撤退時強化：維持」設定なら保持

- 各勢力に1つ。制圧で敵全拠点の鼎を3つ破壊

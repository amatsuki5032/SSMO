# M6-6: バランス調整

読むファイル: GameConfig.cs, WeaponData.cs, DamageCalculator.cs, combat-spec.md
変更ファイル: GameConfig.cs, WeaponData.cs

統合テスト結果をもとにゲームバランスを調整する。

## 調整対象

### 1. 武器種バランス

| 武器種 | 調整項目 |
|--------|---------|
| 大剣 | 火力と攻撃速度のバランス。遅すぎると使われない |
| 双剣 | 手数型なのでDPSが他と同等になるよう倍率調整 |
| 槍 | リーチの優位性と火力のトレードオフ |
| 戟 | 範囲攻撃の強さとリスクのバランス |
| 拳 | 接近リスクに見合うDPS。移動速度でカバー |
| 弓 | 遠距離の安全性と近接時の脆さのバランス |

各武器のN1-N6、C1-C6の倍率・持続時間・ヒットボックスサイズを微調整。

### 2. ダメージ計算パラメータ

- DEFAULT_ATK / DEFAULT_DEF / DEFAULT_MAX_HP の基準値
- 根性補正の閾値（現在: 50%/20%）と補正値（÷1.5/÷2）
- 空中補正（÷2）の妥当性
- クリティカル率（5%）と倍率（×1.5）

### 3. 属性バランス

- 各属性の倍率テーブル（炎0.175/氷0.25/雷0.50/風0.50）
- 氷の凍結確率テーブル
- 雷の感電確率テーブル
- 燃焼の持続ダメージ量
- 斬の最低保証ダメージ

### 4. 試合テンポ

- MATCH_TIME_SECONDS（現在300秒＝5分）
- NPC兵士: HP / ATK / スポーン間隔 / 最大数
- 拠点制圧速度（BASE_CAPTURE_TIME）
- 仙箪ドロップ率
- 無双ゲージの増加速度（攻撃/被弾/チャージ）

### 5. EGバランス

- EG準備時間（EG_CHARGE_SEC: 現在1.0秒）
- EGカウンターの無双消費量（EG_COUNTER_MUSOU_COST）
- EG維持の無双消費速度（EG_MUSOU_DRAIN_RATE）

## テスト方法
- ParrelSync 2人対戦で各武器種の感触確認
- DebugTestHelper（F3:無双MAX、F7:HP20%、F10:アーマー変更）活用
- 各数値変更後にリプレイして比較

## 注意事項
- combat-spec.md の仮値（※仮値と記載）を実測値に更新すること
- GameConfig.cs の定数を変更する場合はコメントに変更理由を記載

# M4-1c: 武器種別ヒットボックス

読むファイル: WeaponData.cs, HitboxData.cs, HitboxSystem.cs
変更ファイル: WeaponData.cs, HitboxData.cs

武器種ごとのヒットボックスパラメータを設定する。

1. WeaponData.cs に追加：
   - 武器種ごとの HitboxData テーブル（N1-N6, C1-C6, D, JA, JC）
   - 大剣: 広範囲・低速（Radius大、Length大）
   - 双剣: 狭範囲・高速（Radius小、ActiveFrame多段）
   - 槍: 前方特化（Length最大、Radius小）
   - 戟: 中範囲・回転系（Radius中、Length中）
   - 拳: 超近距離（Length最小、ActiveFrame最多）
   - 弓: 投射物扱い（将来。仮でN攻撃は近接）

2. HitboxData.cs：
   - GetHitboxData() が WeaponType を受け取り、武器種別データを返すように修正
   - 既存の大剣データを WeaponData に移行

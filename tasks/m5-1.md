# M5-1: Firebase Auth 認証

読むファイル: GameConfig.cs, CLAUDE.md
変更ファイル: GameConfig.cs, 新規 Scripts/Netcode/AuthManager.cs

Firebase Authenticationによるプレイヤー認証を実装する。

1. 新規 Scripts/Netcode/AuthManager.cs：
   - MonoBehaviour（シングルトン）
   - Firebase Auth SDK初期化
   - 匿名認証（Anonymous Auth）でまず実装（メール/パスワードは後）
   - ログイン成功時にUIDを保持
   - NetworkManager接続前にAuth完了を必須にする
   - 認証トークンをサーバーに送信（ServerRpcで検証）

2. 認証フロー：
   - アプリ起動 → Firebase Auth ログイン → UID取得
   - NetworkManager.StartClient() の前に認証完了を確認
   - サーバー側で接続時にUID検証

3. GameConfig.cs に追加：
   - AUTH_TIMEOUT = 10f（認証タイムアウト秒数）

4. Unity設定（手動）：
   - Firebase SDK for Unity をインポート
   - google-services.json / GoogleService-Info.plist を配置
   - Firebase Console でAnonymous Auth を有効化

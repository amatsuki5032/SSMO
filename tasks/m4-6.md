# M4-6: 刻印システム

読むファイル: WeaponData.cs, ComboSystem.cs, HitboxData.cs, GameConfig.cs
変更ファイル: WeaponData.cs, ComboSystem.cs, HitboxData.cs, CharacterState.cs, GameConfig.cs

刻印システム（C1/C6モーション選択）を実装する。

1. CharacterState.cs に追加：
   - enum InscriptionType : byte { Thrust, Formation, Crush, Shield, Conquer, Guard }
   - （突/陣/砕/盾/覇/衛）

2. WeaponData.cs に追加：
   - 刻印ごとのC1パラメータ（4種: 突/陣/砕/盾）
   - 刻印ごとのC6パラメータ（6種: 全種）
   - 各刻印の倍率・ヒットボックス・効果（仮値。具体モーションは未確定事項）
     - 突: 突進系
     - 陣: 範囲系
     - 砕: 高威力単体
     - 盾: 防御付き攻撃
     - 覇: C6専用・超範囲
     - 衛: C6専用・カウンター系

3. ComboSystem.cs：
   - プレイヤーの装備刻印を保持（NetworkVariable<byte> c1Inscription, c6Inscription）
   - TryStartCharge() で C1/C6 の場合に刻印タイプに応じたパラメータを使用

4. HitboxData.cs：
   - GetHitboxData() に刻印タイプを考慮したC1/C6データ返却

5. GameConfig.cs：
   - 各刻印の仮パラメータ（倍率・持続時間・範囲）

※ 刻印選択UIはM6で実装。M4では刻印の効果をコードで切り替え可能にするところまで。

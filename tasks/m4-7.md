# M4-7: 究極強化

読むファイル: EnhancementRing.cs, ComboSystem.cs, GameConfig.cs, CharacterStateMachine.cs
変更ファイル: EnhancementRing.cs, GameConfig.cs, 新規 Scripts/Combat/UltimateEnhancement.cs

究極強化システムを実装する。

1. 新規 Scripts/Combat/UltimateEnhancement.cs：
   - NetworkBehaviour, サーバー権威
   - 連撃強化3回達成後、4回目以降の仙箪7個で究極強化発動
   - **30秒間の超強力バフ**
   - NetworkVariable<bool> isUltimateActive
   - NetworkVariable<float> ultimateRemainingTime
   - 武器種ごとに効果が異なる（仮: 全武器共通で実装。個別効果はM6以降）
   - 共通効果（仮値）:
     - ATK × 1.5
     - DEF × 1.5
     - 移動速度 × 1.2
     - アーマーレベル +1
   - 30秒経過で効果解除
   - 死亡時にリセット

2. EnhancementRing.cs：
   - 連撃Lv3達成後、スロット7が「究極強化」に変化
   - 究極強化発動時に UltimateEnhancement.Activate() を呼ぶ

3. GameConfig.cs に追加：
   - ULTIMATE_DURATION = 30f
   - ULTIMATE_ATK_MULT = 1.5f
   - ULTIMATE_DEF_MULT = 1.5f
   - ULTIMATE_SPEED_MULT = 1.2f

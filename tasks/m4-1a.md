# M4-1a: WeaponData 基盤

読むファイル: GameConfig.cs, CharacterState.cs, ComboSystem.cs
変更ファイル: CharacterState.cs, GameConfig.cs, 新規 Scripts/Shared/WeaponData.cs

武器種ごとのパラメータを管理するデータ構造を作成する。

1. CharacterState.cs の WeaponType enum を確認（GreatSword, DualBlades, Spear, Halberd, Fists, Bow）

2. 新規 Scripts/Shared/WeaponData.cs：
   - static クラス（ScriptableObject は使わない。定数データなので）
   - 武器種ごとのパラメータを定義:

```
WeaponParams 構造体:
  - MoveSpeed: float（移動速度）
  - DashSpeed: float（ダッシュ速度）
  - JumpHeight: float（ジャンプ高さ）
  - AirTime: float（滞空時間）
  - AttackRange: float（基本リーチ）
  - AttackSpeed: float（攻撃速度倍率、1.0基準）
  - float[] NormalMultipliers（N1-N6のモーション倍率）
  - float[] NormalDurations（N1-N6の持続時間）
  - float[] ChargeMultipliers（C1-C6のモーション倍率）
  - float[] ChargeDurations（C1-C6の持続時間）
  - float DashAttackMultiplier
  - float DashAttackDuration
  - float JumpAttackMultiplier
  - float JumpChargeMultiplier
```

   - static GetWeaponParams(WeaponType) → WeaponParams を返す
   - 初期データは大剣(GreatSword)のみ本格的に設定。他5種は大剣ベースで倍率違いの仮値

3. GameConfig.cs：
   - 大剣の既存定数（N1_DURATION等）はWeaponDataに移行
   - GameConfig側は DEFAULT_ プレフィックスで残すか、WeaponData参照に切り替え

combat-spec.md セクション21「武器種」の表を参照:
- 大剣: 射程3m, 速度遅, 威力★★★★★, ジャンプ低, 移動遅
- 双剣: 射程1.5m, 速度速, 威力★★, ジャンプ高, 移動速
- 槍: 射程4.5m, 速度中, 威力★★★, ジャンプ中, 移動中
- 戟: 射程3.5m, 速度中, 威力★★★★, ジャンプ中, 移動中
- 拳: 射程1m, 速度最速, 威力★★, ジャンプ最高, 移動最速
- 弓: 射程100m, 速度遅, 威力★★★, ジャンプ低, 移動中

# SSMO - 真三國無双Online風 4v4アクション対戦ゲーム

## 概要
近接アクションを軸とした 4v4 チーム対戦ゲーム。  
真三國無双Onlineの戦闘体験を現代のネットコード技術で再構築する。

## 技術スタック
| 項目 | 技術 |
|------|------|
| エンジン | Unity 6.3 LTS (C#) |
| ネットワーク | Netcode for GameObjects (NGO) |
| サーバー | Dedicated Server (ヘッドレス) |
| 認証/DB | Firebase Auth / Firestore |
| ホスティング | AWS or GCP (将来) |

## アーキテクチャ
- **サーバー権威型**: ゲーム状態の正解はサーバーが持つ
- **クライアント予測**: 入力は即座にローカル実行、ズレたらサーバー結果で補正
- **ラグコンペンセーション**: 攻撃時にサーバーが「その瞬間」の世界に巻き戻して判定
- **固定ティック 60Hz**: ゲームロジックは FixedUpdate 60Hz / 描画FPSは可変

## プロジェクト構造
```
Assets/
├── Scripts/
│   ├── Netcode/      # ネットワーク同期・予測・補間
│   ├── Combat/       # ヒット判定・ダメージ・コンボ
│   ├── Character/    # 移動・ステート・アニメーション
│   ├── UI/           # HUD・メニュー・ロビー
│   ├── Shared/       # 定数・計算式・データ定義
│   └── Server/       # サーバー専用ロジック・AI
├── Prefabs/
├── Scenes/
├── Models/
├── Materials/
└── Effects/
```
